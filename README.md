# YouTube English Duolingo (Real B)

YouTube URL → local files (./data) → segment learning UI.

## Endpoints

- `POST /api/import` `{url}` → `{jobId, courseId}`
- `GET /api/jobs/:jobId` → `{status, progress, error?, courseId, step?}`
- `GET /api/courses/:courseId` → `{course}` (reads `./data/courses/<courseId>/manifest.json`)

## Local storage layout

After an import succeeds:

```
./data/
  courses/<courseId>/
    manifest.json
    segments.json
    01_transcript.jsonl
    01_transcript.srt
    02_notes_segment_*.md          (generated by ytbnotes)
    03_notes_full_bilingual.md     (generated by ytbnotes)
  jobs/<jobId>.json
```

## Dependencies

Required on your machine:

- `yt-dlp`
- `ffmpeg` (recommended; yt-dlp may use it)
- `OPENAI_API_KEY` (used by `ytbnotes` for Whisper transcription + note generation)

Install (macOS):

```bash
brew install yt-dlp ffmpeg
export OPENAI_API_KEY=...  # put into your shell profile
```

## Run

```bash
npm install
npm run dev
```

Open:
- <http://localhost:3000/import>

## Demo

1) Go to `/import`, paste a YouTube URL, click **Import**.
2) You will be redirected to `/jobs/<jobId>` and see progress.
3) When ready, it auto-navigates to `/courses/<courseId>`.
4) Click **Continue** to start learning.

## Failure handling

If prerequisites are missing (e.g. `yt-dlp`, `OPENAI_API_KEY`) or downloading/transcription fails:

- job becomes `status=error`
- UI shows the error text
- click **Retry** to go back to import page

## Notes / MVP choices

- No database: jobs + courses are stored as local JSON files under `./data/`.
- Segments are created from real transcript lines (default ~30s chunks; configurable).
- Exercises are generated with simple rules (cloze MCQ), no LLM dependency.
- Audio playback uses a YouTube embed with `start=<segment.startSec>` (no audio slicing).
